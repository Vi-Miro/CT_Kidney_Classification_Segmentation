Инструкция для использовани UI классификатора почек
------------------------------------------------------------------------------------------------------------------------

*Эта программа предназначена только для образовательных целей и не может быть использована для медицинской диагностики


------------------------------------------------------------------------------------------------------------------------
Для получения отчета необходимо выполнить следующие действия:

1. Загрузите инетересуемый срез в формате DICOM, при помощи нажатии кнопки "Загрузить DICOM".
* При загрузке DICOM работает модель детекции на наличие почек на изображении для предотвращения ошибочной загрузки
в случае ошибачного срабатывания предупреждения закройте диалоговое окно нажатием кнопки "да"
2. Нажмите кнопку "Запустить анализ" для получения результата работы модели и формирования отчета.
3. Протегируйте DICOM получившимся результатом - для этого нужно нажать кнопку  "Сохранить DICOM с результатами".
4. Отправьте отчет в кафу нажатием на "Отправить в медицинскую систему (Kafka)"

------------------------------------------------------------------------------------------------------------------------
Так же есть дополнительная возможность проверить существующие DICOM теги нажатием на "Показать DICOM теги" 
и сохранить результат на локально при помощи кнопки "Сохранить результат в JSON"
------------------------------------------------------------------------------------------------------------------------
